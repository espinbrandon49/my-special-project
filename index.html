CSS Code Snippet

Do you ever have trouble with CSS? Me too! Here are a few helpful tips that could help!

Flexbox Column


These are helpful properties to make a flexbox column 

.column {
  display: flex;  
  flex-direction: column;  
}

CSS Grid Layout


These are helpful properties to make a CSS Grid Layout 

.grid {
  display: grid;  
  width: 100%;  
  grid-template-columns: repeat(12, 1fr);  
}

Box Border


These are helpful properties to make a simple border 

.border {
  border: 5px dotted goldenrod;
}

Center content in a box


These are helpful properties to center content

.center {
  display: flex;
  justify-content: center;
  align-items: center;  
}

Overlay Card with Title


These are helpful properties to make elements overlap 

.header {
  positon: relative;  
  margin-top: -20px 
}

Change the element theme


These are helpful properties to change element themes 

.theme {
  font-family:  font-family: Impact,
    'Arial Narrow Bold', sans-serif;  
  color: red;  
  background-color: black;
  transform: rotateX(35deg) 
}
